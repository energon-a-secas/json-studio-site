# JSON Builder

A visual, block-based JSON editor hosted on GitHub Pages. No install, no build step.

Drag blocks from the left palette onto the canvas to build any JSON structure. Each node is editable inline: rename keys, set values, drag to reorder, and collapse subtrees. The right panel shows a live syntax-highlighted preview with copy and download buttons. You can also paste raw JSON directly via Edit mode, or import a `.json` file.
